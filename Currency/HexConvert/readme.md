## 环境需求
.ini 文件解析器
```bash
pip install configparser
```

重新编译 .exe 在 cmd 中执行
```bash
pip install pyinstaller
pyinstaller --onefile HexConvert.py
```

重新生成的可执行文件在生成的 dist 文件夹
